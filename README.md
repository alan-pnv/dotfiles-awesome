# dotfiles-awesome
My dotfiles for awesome wm

## Installation
```sh
	git clone https://github.com/alan-pnv/dotfiles-awesome.git
	cd dotfiles-awesome
	cp -r .bashrc .profile .vim .vimrc ~
	cp -r awesome terminator ~/.config/
	cp bin/* ~/.local/bin
```
